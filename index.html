CSS Flexbox Reference

Activate Flexbox

Property Value Description
display: flex; Defines the element as a flex container
inline-flex; Allows flex containers to become inline elements

justify-content

Property Value Description
justify-content: flex-start; all items on the main axis will be positioned in order from the left of the parent container with no extra space between or before them
justify-content: flex-end; all items on the main axis will be positioned in order from the right of the parent container with no extra space between or before them
justify-content: center; all items on the main axis will be positioned in order in the center of the parent container with no extra space between, before, or after them
justify-content: space-around; all items will be positioned with equal space before and after each item with 2x the space between elements
justify-content: space-between; all items will be positioned with equal space between them without extra space before the first element or after the last element

align-items

Property Value Description
align-items: flex-start; all elements will be positioned at the top of the cross axis
align-items: flex-end; all elements will be positioned at the bottom of the cross axis
align-items: center; all elements will be positioned halfway between the top and the bottom of the cross axis
align-items: baseline; the bottom of the elements will be aligned with each other
align-items: stretch; default value. elements will stretch from top to bottom if height is not specified or a min-height has been specified

flex

Property Value Description
flex-grow: #; 0 is the default value. specifies which elements will grow and in what proportions
flex-shrink: #; 1 is the default value. specifies which elements will shrink and in what proportions
flex-basis: #px; specifies the element's width before it stretches or shrinks
flex: # # #px; a shorthand property for declaring flex-grow, flex-shrink, and flex-basis in that order. flex can be set to 2 values for flex-grow and flex-basis, but not flex-shrink and flex-basis

flex-wrap

Property Value Description
flex-wrap: wrap; child elements of a flex container will be moved to a new line if it goes past the set width
flex-wrap: wrap-reverse; works the same as wrap, but the order of the rows will be reversed
flex-wrap: nowrap; default value. prevents the items from wrapping

align-content

Property Value Description
align-content: flex-start; all rows will be positioned at the top of the cross axis with no extra space between
align-content: flex-end; all rows will be positioned at the bottom of the cross axis with no extra space between
align-content: center; all rows will be positioned at the center of the cross axis with no extra space between
align-content: space-between; all rows will be positioned evenly from the top to the bottom of the cross axis with no extra space before the first row and after the last row
align-content: space-around; all rows will be positioned evenly from top to the bottom of the cross axis with extra space at the top and the bottom and between each row
align-content: stretch; default value. if min-height or no height is specified, the rows will stretch to fill the parent container from top to bottom

flex-direction

Property Value Description
flex-direction: row; default value. elements will be positioned from left to right across the container starting at the top left corner
flex-direction: row-reverse; elements will be positioned from right to left across the container starting at the top right corner
flex-direction: column; elements will be positioned from top to bottom of the container starting from the top left corner
flex-direction: column-reverse; elements will be positioned from the bottom to top of the container starting from the bottom left corner

flex-flow

Property Value Description
flex-flow: flex-wrap flex-direction; shorthand property used to declare the flex-flow and flex-direction of an element
